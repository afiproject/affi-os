# SLOTY — 時間共有マーケット

時間をスロット単位で売買できるマーケットプレイスアプリ。
「すれ違い」で今ヒマな人と即マッチ。チケット制でシンプルな課金体験。

## デモモード（UI確認用）

DB・認証・Stripe の設定不要でUI/動線を確認できます。

### ローカル起動

```bash
cp .env.example .env.local
npm install
npm run dev
```

ブラウザで http://localhost:3000 を開く（スマホ幅で確認推奨）。

### Vercel デプロイ手順

1. GitHub にリポジトリを push（済み）
2. [vercel.com](https://vercel.com) にログイン → **Add New Project**
3. GitHub リポジトリを選択してインポート
4. **Environment Variables** に `NEXT_PUBLIC_DEMO_MODE` = `1` を追加
5. **Deploy** をクリック
6. デプロイ完了後、発行された URL をスマホで開いて確認

## デモで確認できる操作一覧

1. **起動画面** — 2秒のグラデーションスプラッシュ（青紫→ラベンダー→ピンク）
2. **プロフィール設定** — /profile でTinder風写真カード、目的タグ・趣味タグ・職業・年収レンジ・生活タグを設定
3. **ダークモード** — /profile のトグルで切替（localStorage保存）
4. **すれ違い** — /nearby で「イマヒマON」→ 🌍マップで近くの人を表示 → ピン送信（5🎫）→ /pings で承認 → 予約成立
5. **AI待ち合わせ提案** — 対面ピン送信/承認時にAIが公共の場所を3件提案（仙台駅周辺POI）
6. **カレンダー** — /calendar でフレンド予定オーバーレイ（👥ボタン）、共通空き時間表示、日セルダブルタップでモーダル
7. **スロット出品→購入** — /market/sell で出品 → /market で検索 → 詳細で予約（🎫消費）→ /bookings で確認
8. **広場→依頼→成立** — /square で投稿（2🎫）→ 他の投稿で「依頼」（5🎫）→ /requests/inbox で承認
9. **非公開予定＋衝突チェック** — /calendar で予定追加（種別: busy/free/private/buffer）→ /market/sell で同時間に出品試行 → エラー
10. **チケット消費** — 投稿(2🎫)・依頼(5🎫)・ピン(5🎫)でリアルに減算、/profile でチャージ可能
11. **写真ガード** — 写真未登録でチェックイン・投稿・依頼・ピン送信が不可

## 設計方針

- **チケット制**: UIに現金(¥)表記は一切なし。全てチケット🎫で統一
- **時間が主役**: カレンダー→空き時間→マッチが基本フロー
- **AI提案層**: `src/lib/ai.ts` にルールベース実装。将来LLM差し替え可能
